# English Learning App 🗣️

A React-based web application designed to help users improve their English speaking and listening skills through interactive exercises.

## Features

### 🎯 Pronunciation Practice
- Speech recognition technology for real-time feedback
- Scoring system based on pronunciation accuracy
- Practice with common English phrases and tongue twisters

### 👂 Listening Comprehension
- Audio-based exercises with multiple-choice questions
- Interactive quizzes to test understanding
- Immediate feedback and scoring

### 🗣️ Speaking Exercises
- Guided speaking prompts for different scenarios
- Audio recording capabilities
- Speech-to-text transcription
- Practice conversation, description, opinion sharing, and storytelling

### 📊 Progress Tracking
- Track scores and improvement over time
- Monitor learning streaks
- View detailed statistics and achievements
- Local storage persistence

## Technology Stack

- **React 18** with TypeScript
- **Vite** for fast development and building
- **Web Speech API** for speech recognition
- **MediaRecorder API** for audio recording
- **Local Storage** for progress persistence
- **CSS3** for modern styling

## Getting Started

### Prerequisites
- Node.js (version 16 or higher)
- npm or yarn package manager

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to `http://localhost:3000`

### Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
- `npm run typecheck` - Run TypeScript compiler check

## Browser Compatibility

This app uses modern web APIs and requires:
- Chrome 25+ (recommended)
- Firefox 44+
- Safari 14.1+
- Edge 79+

**Note**: Speech recognition works best in Chrome and Edge browsers.

## Usage

### Pronunciation Practice
1. Click on "Pronunciation Practice" from the home screen
2. Read the displayed phrase aloud
3. Click "Start Speaking" and speak clearly
4. Get instant feedback on your pronunciation accuracy

### Listening Exercises
1. Select "Listening Comprehension"
2. Play the audio clip
3. Answer the multiple-choice questions
4. Review your results and try again if needed

### Speaking Practice
1. Choose "Speaking Exercises"
2. Read the prompt and any helpful tips
3. Record your response using the built-in recorder
4. Review your transcript and recording

### Progress Tracking
1. Navigate to "Progress Tracking"
2. View your scores, streaks, and statistics
3. Monitor your improvement over time

## Privacy & Data

- All data is stored locally in your browser
- No personal information is transmitted to external servers
- Audio recordings are processed locally and not stored permanently
- You can reset your progress at any time
